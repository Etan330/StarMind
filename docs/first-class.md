# 抖音/小红书/B站 内容大类与细分垂类调研
本文件整合三大平台官方分区与主流内容生态，汇总为8大通用内容大类及对应中粒度细分垂类，适配跨平台内容调研使用。

## 一、知识学习类
**核心属性**：信息输出、认知提升、技能教学，三大平台共同发力的泛知识赛道
- 科学科普：自然科学、冷知识、天文地理、生物医学常识
- 商业财经：金融理财、商业分析、创业副业、行业洞察
- 职场教育：职业技能、求职面试、职场经验、考学考证
- 人文社科：历史文化、法律科普、心理学、社会观察
- AI与科技：AI工具教程、编程技术、互联网行业观察
- 语言学习：外语教学、口语提升、应试语言技巧

## 二、生活方式类
**核心属性**：日常记录、生活技巧、消费种草，抖音、小红书的核心流量赛道
- 日常记录：Vlog、独居/情侣/家庭日常、通勤生活
- 生活技巧：收纳整理、居家妙招、省钱攻略、时间管理
- 美食饮品：菜谱教程、探店打卡、食材测评、饮品制作
- 旅行户外：旅行攻略、景点打卡、露营徒步、户外生活
- 家居家装：装修设计、软装搭配、家电选购、租房改造
- 萌宠动物：宠物日常、养宠科普、宠物训练
- 情感心理：恋爱婚姻、情绪管理、人际关系、情感共鸣
- 健康养生：健身减脂、养生调理、睡眠饮食改善
- 三农乡村：乡村生活、农业种养、新农人日常

## 三、娱乐休闲类
**核心属性**：情绪消遣、泛娱乐内容，抖音、B站的流量基本盘
- 游戏内容：游戏解说、攻略教程、赛事资讯、游戏整活
- 影视综艺：影视解说、剧集吐槽、综艺剪辑、二创混剪
- 音乐舞蹈：歌曲翻唱、乐器演奏、舞蹈翻跳、音乐创作
- 搞笑剧情：段子短剧、反转剧情、搞笑日常、街头采访
- 明星娱乐：明星资讯、八卦吃瓜、追星物料
- 二次元：番剧动漫、ACG二创、cosplay、手办分享

## 四、时尚美护类
**核心属性**：颜值经济、变美种草，小红书的标杆优势赛道
- 美妆护肤：化妆教程、护肤干货、彩妆试色、产品测评
- 时尚穿搭：OOTD、穿搭技巧、风格定位、单品种草
- 造型配饰：发型美甲、珠宝配饰、鞋包推荐
- 医美美体：医美科普、身材管理、美体教程

## 五、科技数码类
**核心属性**：3C产品、技术分享，男性用户占比偏高的垂类赛道
- 数码测评：手机、电脑、耳机等3C产品深度测评
- 数码教程：软件技巧、设备使用指南、数码开箱
- 硬件DIY：电脑装机、外设改装、数码配件
- 行业资讯：数码行业动态、前沿黑科技解读

## 六、母婴亲子类
**核心属性**：垂直家庭消费赛道，用户粘性与转化意愿强
- 育儿科普：早教知识、喂养攻略、育儿理念
- 萌娃日常：宝宝成长记录、亲子互动日常
- 母婴好物：奶粉、童装、母婴用品测评种草
- 孕产经验：孕期记录、产后恢复、备孕知识

## 七、汽车房产类
**核心属性**：高客单价垂类，以资讯、测评、攻略为主
- 汽车内容：新车测评、购车攻略、用车技巧、改装玩车
- 房产内容：楼市资讯、买房攻略、租房经验、避坑指南

## 八、艺术文化类
**核心属性**：小众高粘性赛道，主打审美提升与文化兴趣
- 艺术设计：绘画教程、设计干货、审美提升、摄影技巧
- 传统文化：非遗文化、国学经典、书法戏曲
- 文艺分享：书籍推荐、电影解读、展览打卡、手工手作
